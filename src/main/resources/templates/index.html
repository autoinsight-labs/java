Sobre o Sistema

O AutoInsight é um sistema desenvolvido em Spring Boot com Thymeleaf para gerenciamento completo de reservas e veículos.

Funcionalidades:
  • Gestão de Usuários
  • Cadastro de Veículos
  • Controle de Reservas
  • Sistema de Roles
  • Interface Responsiva
